## Prerequisites

<Callout kind="info" title="Before You Begin">
Ensure you have:
- Node.js `{version >= 14}` installed
- A code editor like VS Code
- Access to a terminal or command prompt
</Callout>

Review your Zohocorp developer account status. If you don't have one, proceed to the next section.

## Create a Developer Account

Sign up for a free Zohocorp developer account to access APIs and tools.

<Steps>
  <Step title="Visit Dashboard" icon="monitor">
    Navigate to `https://dashboard.example.com/signup`.
  </Step>
  <Step title="Register" icon="user-plus">
    Enter your email and create a password.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox and click the verification link.
  </Step>
  <Step title="Generate API Key" icon="key">
    Go to Account Settings > API Keys and create a new key named `Quickstart Key`.
  </Step>
</Steps>

<Callout kind="tip">
Save your `{API_KEY}` securely. You'll need it for all requests.
</Callout>

## Install the Zohocorp CLI

The CLI helps you scaffold client scripts and widgets quickly.

<Tabs>
  <Tab title="macOS" icon="apple">
````bash
brew install zohocorp/tap/zohocorp-cli
````
  </Tab>
  <Tab title="Windows" icon="monitor">
````bash
winget install Zohocorp.CLI
````
  </Tab>
  <Tab title="Linux / npm" icon="terminal">
````bash
npm install -g @zohocorp/cli
````
  </Tab>
</Tabs>

Verify installation:

````bash
zohocorp --version
````

## Make Your First API Call

Test connectivity by fetching a list of records from your CRM module.

Set your `{API_KEY}` as an environment variable:

````bash
export ZOHO_API_KEY=your-api-key-here
````

<Request tabs="JavaScript,cURL" show-lines="true">
````javascript
const response = await fetch('https://api.example.com/crm/v6/Leads', {
  headers: {
    'Authorization': `Zoho-oauthtoken ${process.env.ZOHO_API_KEY}`
  }
});
const data = await response.json();
console.log(data);
````
````bash
curl -H "Authorization: Zoho-oauthtoken $ZOHO_API_KEY" \
  https://api.example.com/crm/v6/Leads
````
</Request>

<Response tabs="200">
````json
{
  "data": [
    {
      "id": "123456789",
      "First_Name": "John",
      "Last_Name": "Doe",
      "Email": "john.doe@example.com"
    }
  ]
}
````
</Response>

## Build Your First Client Script

Client scripts extend Zohocorp CRM forms with custom logic.

<Steps>
  <Step title="Scaffold Script" icon="file-plus">
    Run the CLI command:
    
````bash
zohocorp script create my-first-script --module Leads
````
  </Step>
  <Step title="Edit Script" icon="edit-3">
    Open `my-first-script/index.js` and add:
    
````javascript
function validateLead() {
  const email = zohocrm.ui.getFieldValue('Email');
  if (!email.includes('@')) {
    zohocrm.ui.showError('Invalid email format');
    return false;
  }
  return true;
}

zohocrm.events.on('before_save', validateLead);
````
  </Step>
  <Step title="Deploy" icon="upload-cloud">
````bash
zohocorp script deploy my-first-script
````
  </Step>
</Steps>

Navigate to a Lead record in your CRM to test the validation.
