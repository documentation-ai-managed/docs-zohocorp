## Overview

Zohocorp empowers developers to customize and extend Zoho CRM functionality. You can build powerful integrations using robust APIs, create dynamic client scripts, develop reusable widgets, and automate workflows with functions. This platform provides all the tools you need to tailor CRM to your business requirements.

## Quick Start

Get up and running with Zohocorp in minutes.

<Steps>
  <Step title="Sign Up and Get API Key" icon="key">
    Create a Zohocorp developer account at `https://dashboard.zohocorp.com`. Generate your API key from the developer console.

    <CodeGroup tabs="JavaScript,cURL">
      ```javascript
      const API_KEY = 'YOUR_API_KEY';
      const headers = { Authorization: `Zoho-oauthtoken ${API_KEY}` };
      ```
      ```bash
      export API_KEY='YOUR_API_KEY'
      curl -H "Authorization: Zoho-oauthtoken $API_KEY" https://api.zohocorp.com/crm/v6/Leads
      ```
    </CodeGroup>
  </Step>

  <Step title="Make Your First API Call" icon="zap">
    Fetch Leads data using the CRM API endpoint.
  </Step>

  <Step title="Explore Extensions" icon="settings">
    Dive into client scripts or widgets for UI customizations.
  </Step>
</Steps>
