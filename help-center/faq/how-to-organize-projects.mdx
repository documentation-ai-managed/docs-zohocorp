## Overview

You can organize projects in Zohocorp using Spaces, Folders, and Pages.

## Steps

1. Create a new Space for each project.
2. Add Folders to group related topics.
3. Create Pages for specific documentation.

**Tip:** Use naming conventions for clarity.

## Example

- Space: "Marketing Project"
- Folder: "Campaigns"
- Page: "Email Launch Plan"
